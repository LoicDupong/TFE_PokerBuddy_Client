# PokerBuddy ♠️  
**Home poker nights, made simple.**

PokerBuddy is a **mobile-first web application** designed to help players **organize, run, and track home poker sessions** — before, during, and after game night.

---

## Concept

Home poker games are often managed with group chats, spreadsheets, or paper notes.  
PokerBuddy centralizes everything into **one clean, intuitive app**, built for real-life poker tables.

---

## Features (V1)

### 🗓️ Session management
- Create and manage poker nights
- View upcoming and past sessions
- Manage players and results per session

### ⏱️ In-game tools
- Poker-friendly timer
- Flow adapted to live home games
- Designed for quick interactions on mobile

### 📊 History & statistics
- Session history
- Player results and winnings
- Performance overview

### 📖 Poker rules
- Built-in rules reference for all players
